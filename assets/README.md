# Assets - Images and Media

This directory contains image files and media assets for the MedQuest app.

## Contents

- **Screenshots** (*.png) - Screenshots from exams and UI mockups
- **Icons** - App icons and branding elements
- **Diagrams** - Medical diagrams and visual references

## Usage

These assets are used for:
- App documentation and README
- UI previews and demos
- Question images (when available)
- Marketing and presentation materials

Note: Question images from PDFs are stored in `../data/raw/pdfs/`
