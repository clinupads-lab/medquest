================================================================================
                           COMO USAR NO APP
================================================================================

1. IMPORTAR AS QUESTOES:

   import { ALL_QUESTIONS } from './data/processed/questions';

2. USAR NO COMPONENTE:

   const [questions] = useState(ALL_QUESTIONS);
   const [currentIndex, setCurrentIndex] = useState(0);

   const question = questions[currentIndex];

3. FILTRAR:

   // Por banca
   const iamspe = ALL_QUESTIONS.filter(q => q.banca === 'IAMSPE');

   // Por disciplina
   const clinica = ALL_QUESTIONS.filter(q => q.subject === 'Clínica Médica');

   // Por ano
   const recent = ALL_QUESTIONS.filter(q => q.year >= 2023);
